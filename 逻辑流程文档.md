# DPI 自动化测试系统逻辑流程文档

## 一、总体执行流程

### 1.1 主流程概览

```mermaid
graph TD
    Start([程序启动]) --> A[解析命令行参数]
    A --> B{是否生成脚本?}
    B -->|是 -bat| C[生成 BAT 脚本]
    B -->|是 -ps1| D[生成 PS1 脚本]
    B -->|否| E[解析 Excel 用例]
    C --> End1([结束])
    D --> End1
    E --> F[读取配置信息]
    F --> G[连接 DPI 设备]
    G --> H{遍历 Sheet}
    H --> H1[创建 DynamicFileHandler]
    H1 --> I{遍历用例}
    I --> J{检查执行状态}
    J -->|跳过| I
    J -->|执行| K{判断安装类型}
    K -->|全新安装| L[执行全新安装]
    K -->|模式切换| M[执行模式切换]
    K -->|升级| N[执行版本升级]
    L --> O[记录测试结果]
    M --> O
    N --> O
    O --> P{还有用例?}
    P -->|是| I
    P -->|否| Q{还有 Sheet?}
    Q -->|是| R[关闭 DynamicFileHandler]
    R --> H
    Q -->|否| S[生成测试报告]
    S --> End2([结束])
```

## 二、版本路径获取流程

### 2.1 FTP 路径获取逻辑

```mermaid
graph LR
    Start([需要版本 X]) --> A[根据模式获取分类]
    A --> B[获取项目列表]
    B --> C{在 versions.json<br/>中查找版本?}
    C -->|找到| D[提取 FTP 路径]
    C -->|未找到| E[触发自动更新]
    D --> F{路径是否为空?}
    F -->|否| G[返回 FTP 路径]
    F -->|是| H[抛出异常:<br/>未找到程序包]
    E --> I[从 RDM 平台提取版本信息]
    I --> J[更新 versions.json]
    J --> K{再次查找版本?}
    K -->|找到| D
    K -->|未找到| L[抛出异常:<br/>版本不存在]
    G --> End([结束])
    H --> End
    L --> End
```

### 2.2 模式到分类映射

```mermaid
graph LR
    A[DPI 模式] --> B{提取模式后缀}
    B -->|is/isbns| C[信息安全执行单元]
    B -->|bns/ns/bnsns| D[网络安全执行单元]
    B -->|ds| E[数据安全执行单元]
    B -->|其他| F[抛出异常:<br/>无法识别的模式]
```

## 三、全新安装流程

### 3.1 全新安装完整流程

```mermaid
graph LR
    Start([开始全新安装]) --> A[获取 FTP 路径]
    A --> B[连接 FTP 服务器]
    B --> C{验证安装包存在?}
    C -->|否| D[抛出异常]
    C -->|是| E{是否扫描已有安装包?}

    E -->|是| F[扫描目标服务器]
    E -->|否| G[下载安装包]

    F --> H{找到已有安装包?}
    H -->|是| I[使用已有安装包]
    H -->|否| G

    G --> J[解压第一层<br/>外层压缩包]
    I --> J

    J --> K{检测版本类型}
    K -->|ACT-DPI-EU-<br/>新版本| L[解压第二层<br/>使用 tar]
    K -->|ACT-DPI-ISE-<br/>旧版本| M[解压第二层<br/>使用 unzip]

    L --> N[定位安装脚本<br/>第2层]
    M --> O[解压第三层<br/>使用 tar]
    O --> P[定位安装脚本<br/>第3层]

    N --> Q[执行 install.sh]
    P --> Q

    Q --> R[执行模式切换]
    R --> S{DPI 启动成功?}
    S -->|是| T[✓ 安装成功]
    S -->|否| U[✗ 安装失败]

    T --> End([结束])
    U --> End
    D --> End
```

### 3.2 安装包解压逻辑

```mermaid
graph LR
    Start([开始解压]) --> A{检测安装包前缀}

    A -->|ACT-DPI-EU-| B[新版本 v2<br/>2层压缩]
    A -->|ACT-DPI-ISE-<br/>ACT-DPI-NSE-<br/>ACT-DPI-DSE-| C[旧版本 v1<br/>3层压缩]
    A -->|其他| D[抛出异常]

    B --> E[第1层: unzip<br/>外层压缩包]
    C --> E

    E --> F{版本类型?}
    F -->|新版本| G[第2层: tar -xzf<br/>ALL-BOOT-ALL-ALL-*.tar.gz]
    F -->|旧版本| H[第2层: unzip<br/>*.tar.gz]

    G --> I[安装脚本在第2层<br/>无需第3层]
    H --> J{安装还是升级?}

    J -->|升级| K[升级脚本在第2层<br/>无需第3层]
    J -->|安装| L[第3层: tar -xzf<br/>*.tar.gz]

    L --> M[安装脚本在第3层]

    I --> End([解压完成])
    K --> End
    M --> End
    D --> End
```

## 四、模式切换流程

### 4.1 模式切换完整流程

```mermaid
graph LR
    Start([开始模式切换]) --> A[关闭 agent]
    A --> B[备份当前 DPI]
    B --> C{当前版本是否<br/>匹配源版本?}

    C -->|匹配| D[follow_up_mode = 0<br/>无需操作]
    C -->|不匹配| E{是否优先使用备份?}

    E -->|否| F[follow_up_mode = 2<br/>全新安装]
    E -->|是| G[扫描备份目录]

    G --> H{找到版本匹配的备份?}
    H -->|否| F
    H -->|是| I{备份模式是否匹配?}

    I -->|否| G
    I -->|是| J[follow_up_mode = 1<br/>恢复备份]
    J --> K[复制备份到 /opt/dpi]
    K --> L[执行 ldconfig]

    D --> M{follow_up_mode?}
    F --> N[执行全新安装流程]
    L --> M

    M -->|0| O[跳过安装]
    M -->|1| P[执行模式切换]
    M -->|2| N

    N --> P
    O --> P

    P --> Q[调用 mod_switch]
    Q --> R{模式切换成功?}
    R -->|是| S[✓ 切换成功]
    R -->|否| T[✗ 切换失败]

    S --> End([结束])
    T --> End
```

## 五、版本升级流程

### 5.1 版本升级完整流程

```mermaid
graph LR
    Start([开始版本升级]) --> A[关闭 agent]
    A --> B[备份当前 DPI]
    B --> C[安装源版本<br/>同模式切换流程]
    C --> D{源版本安装成功?}
    D -->|否| E[✗ 升级失败]
    D -->|是| F[预修改 xsa.json]
    F --> G{使用升级系统?}

    G -->|是| H[升级系统接管模式]
    G -->|否| I[本地升级模式]

    H --> J[等待升级系统启动]
    J --> K{升级系统启动成功?}
    K -->|否| L[✗ 升级系统启动超时]
    K -->|是| M[等待升级完成]
    M --> N{升级完成?}
    N -->|否| O[✗ 升级超时]
    N -->|是| P[验证版本号]

    I --> Q[获取目标版本 FTP 路径]
    Q --> R[下载目标版本安装包]
    R --> S[解压安装包]
    S --> T[执行 upms_install.sh]
    T --> U{升级脚本执行成功?}
    U -->|否| V[✗ 升级脚本失败]
    U -->|是| P

    P --> W{版本号是否正确?}
    W -->|是| X[✓ 升级成功]
    W -->|否| Y[✗ 版本号不匹配]

    X --> End([结束])
    Y --> End
    E --> End
    L --> End
    O --> End
    V --> End
```

```mermaid
graph LR

    D -->|是| E[✗ 启动超时]

    B -->|已启动| F[等待升级完成]

    H --> I{超时?}

    G -->|匹配| K[✓ 升级成功]
```
